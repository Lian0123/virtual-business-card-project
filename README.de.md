# Virtual Business Card Project

Erstelle, bearbeite, überprüfe und exportiere professionelle digitale Visitenkarten direkt im Browser. Unterstützt PWA, WebMCP-Fernsteuerung (JSON-RPC) sowie Export als PNG/JPG/WebP/HTML/VCF.

## Hauptfunktionen
- Kartenbearbeitung: Name, Titel, Firma, Kontakt, Adresse, Website, Intro
- Visuelles Design: Theme, Vorlage, Hintergrund, Rahmen, Animation, Sticker
- Export: PNG / JPG / WebP / HTML / VCF
- KI-Integration: WebMCP (`tools/list`, `tools/call`, `card.*`)
