# Virtual Business Card Project

Create, edit, preview, and export professional digital business cards directly in the browser. Supports PWA, WebMCP (JSON-RPC) remote control, and export formats: PNG/JPG/WebP/HTML/VCF.

## Key Features
- Card editing: name, title, company, contacts, address, website, intro
- Visual design: theme, template, background, border, animation, stickers
- Export: PNG / JPG / WebP / HTML / VCF
- AI integration: WebMCP with `tools/list`, `tools/call`, and `card.*` tools
