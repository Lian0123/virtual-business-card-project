# Virtual Business Card Project

Создавайте, редактируйте, просматривайте и экспортируйте профессиональные цифровые визитки прямо в браузере. Поддерживаются PWA, удалённое управление через WebMCP (JSON-RPC) и форматы экспорта PNG/JPG/WebP/HTML/VCF.

## Основные возможности
- Редактирование: имя, должность, компания, контакты, адрес, сайт, описание
- Дизайн: темы, шаблоны, фон, рамки, анимации, стикеры
- Экспорт: PNG / JPG / WebP / HTML / VCF
- Интеграция с AI: WebMCP (`tools/list`, `tools/call`, `card.*`)
