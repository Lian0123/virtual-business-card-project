# Virtual Business Card Project

브라우저에서 전문 디지털 명함을 생성·편집·미리보기·내보내기 할 수 있습니다. PWA, WebMCP(JSON-RPC) 원격 제어, PNG/JPG/WebP/HTML/VCF 내보내기를 지원합니다.

## 주요 기능
- 명함 편집: 이름, 직함, 회사, 연락처, 주소, 웹사이트, 소개
- 시각 디자인: 테마, 템플릿, 배경, 테두리, 애니메이션, 스티커
- 내보내기: PNG / JPG / WebP / HTML / VCF
- AI 연동: WebMCP(`tools/list`, `tools/call`, `card.*`)
