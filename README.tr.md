# Virtual Business Card Project

Tarayıcıda profesyonel dijital kartvizitler oluşturun, düzenleyin, önizleyin ve dışa aktarın. PWA, WebMCP (JSON-RPC) uzaktan kontrolü ve PNG/JPG/WebP/HTML/VCF dışa aktarma formatlarını destekler.

## Temel Özellikler
- Kart düzenleme: ad, unvan, şirket, iletişim, adres, web sitesi, tanıtım
- Görsel tasarım: tema, şablon, arka plan, kenarlık, animasyon, sticker
- Dışa aktarma: PNG / JPG / WebP / HTML / VCF
- AI entegrasyonu: WebMCP (`tools/list`, `tools/call`, `card.*`)
