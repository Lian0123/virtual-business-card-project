## 基本問題

### Q1: 這是免費工具嗎？
**A:** 是的，完全免費！沒有隱藏費用或訂閱要求。

### Q2: 我需要註冊賬戶嗎？
**A:** 不需要。應用程式無需登入即可使用。所有數據都保存在您的瀏覽器中。

### Q3: 支持哪些語言？
**A:** 支援 5 種語言：
- 繁體中文 🇹🇼
- 簡體中文 🇨🇳
- English 🇺🇸
- 日本語 🇯🇵
- 한국어 🇰🇷

可在右上角語言選擇器中切換。

### Q4: 應用程式可以離線使用嗎？
**A:** 可以。第一次加載後，大部分功能可以離線使用。但導出某些格式可能需要網絡連接。

### Q5: 支持哪些操作系統？
**A:** 支援所有主要操作系統：
- Windows
- macOS
- Linux
- iOS
- Android

## 使用問題

### Q6: 如何上傳頭像？
**A:** 步驟：
1. 在「基本資訊」選項卡中找到頭像框
2. 點擊頭像框
3. 選擇本地圖片文件（JPG、PNG、GIF、WebP）
4. 圖片將自動上傳並顯示

### Q7: 如何更改頭像形狀？
**A:** 在「基本資訊」選項卡中，點擊「頭像形狀」按鈕：
- 🔵 圓形 - 現代友善
- ⬜ 方形 - 專業正式
- ⬛ 圓角方形 - 平衡溫和

### Q8: 我可以添加多少個貼圖？
**A:** 無限制。您可以添加任意數量的文字和圖片貼圖。但過多貼圖可能會影響性能。

### Q9: 貼圖支持哪些格式？
**A:** 
- **文字貼圖：** 純文本，支援特殊字符和表情符號
- **圖片貼圖：** JPG、PNG、GIF、WebP

### Q10: 如何調整貼圖的位置和大小？
**A:** 在「貼圖編輯」選項卡中：
1. 選擇您要編輯的貼圖
2. 使用滑杆調整 X 和 Y 位置
3. 使用寬度和高度控制大小（圖片貼圖）
4. 使用大小滑杆調整文字大小

### Q11: 如何刪除貼圖？
**A:** 在「貼圖編輯」選項卡中，點擊要刪除的貼圖旁邊的「刪除」按鈕。

### Q12: 支持多少種頭像大小？
**A:** 支援任何尺寸，但建議：
- 最小：150x150px
- 推薦：300-600px
- 最大：2000x2000px
- 文件大小：< 5MB

## 功能問題

### Q13: 有多少種卡片類型？
**A:** 3 種：
1. **標準卡片** - 展示基本信息
2. **二維碼卡片** - 包含 QR 碼
3. **自訂卡片** - 完全自由設計

### Q14: 動畫效果在所有瀏覽器中都工作嗎？
**A:** 大多數情況下是的。支援所有現代瀏覽器（Chrome、Firefox、Safari、Edge）。Internet Explorer 不支援。

### Q15: 如何控制動畫播放？
**A:** 使用左上角的控制按鈕：
- **▶ 播放** - 開始或恢復動畫
- **⏸ 暫停** - 暫停動畫
- **🔄 重播** - 重新開始動畫

也可以點擊卡片本身來暫停/播放。

### Q16: 支持多少種設計範本？
**A:** 7 種預設範本：
1. 專業風格
2. 現代風格
3. 極簡風格
4. 活力風格
5. 日落風格
6. 海洋風格
7. 森林風格

### Q17: 我可以自訂範本顏色嗎？
**A:** 可以。點擊任何範本應用其基本樣式，然後在「進階設定」中自訂所有顏色。

### Q18: 邊框漸層如何工作？
**A:** 
1. 進入「進階設定」
2. 找到「卡片邊框模式」
3. 選擇「漸層」
4. 選擇漸層色彩 1 和色彩 2
5. 預覽中會看到 135 度角的漸層邊框

### Q19: 可以同時應用多個濾鏡嗎？
**A:** 可以。所有濾鏡都是累積效果，您可以組合使用：
- 亮度 + 對比度 + 飽和度
- 色調 + 銳度 + 模糊
- 等等任何組合

### Q20: 如何重置所有設定？
**A:** 目前沒有一鍵重置功能。解決方案：
- 刷新頁面重新加載默認設定
- 或手動調整每個設定回默認值

## 導出問題

### Q21: 支持哪些導出格式？
**A:** 5 種格式：
1. **PNG** - 無損圖片，支援透明
2. **JPG** - 有損圖片，檔案小
3. **WebP** - 現代格式，最小尺寸
4. **HTML** - 互動網頁版本
5. **VCF** - 聯繫卡片格式

### Q22: 不同格式的最佳用途是什麼？
**A:**
| 格式 | 用途 | 優點 |
|------|------|------|
| PNG | 高質量打印、社交媒體 | 無損、透明 |
| JPG | 常規網絡使用 | 檔案小 |
| WebP | 移動設備 | 最小尺寸 |
| HTML | 網站嵌入、動畫展示 | 互動性強 |
| VCF | 聯繫人導入 | 標準格式 |

### Q23: 導出的文件大小是多少？
**A:** 典型大小：
- PNG：200-500KB
- JPG：50-200KB
- WebP：30-100KB
- HTML：100-300KB
- VCF：< 10KB

### Q24: 如何在 HTML 導出中播放動畫？
**A:** HTML 導出包含完整的動畫：
1. 在瀏覽器中打開 HTML 文件
2. 動畫自動開始播放
3. 點擊卡片可暫停/播放
4. 可嵌入網站 iframe 中

### Q25: 導出的 HTML 可以編輯嗎？
**A:** 可以。HTML 文件是標準格式，可以用任何文本編輯器編輯。但高級編輯需要懂 HTML/CSS。

### Q26: VCF 文件如何使用？
**A:** 
1. 下載 VCF 文件
2. 在手機或通訊錄應用中打開
3. 點擊「添加聯繫人」
4. 信息將自動導入

### Q27: 我可以更改導出的圖片尺寸嗎？
**A:** 目前導出尺寸是固定的。但您可以：
- 使用圖片編輯工具調整大小
- 在 HTML 導出中修改 CSS
- 導出後使用在線工具調整

### Q28: 如何打印我的名片？
**A:** 步驟：
1. 導出為 PNG 或 JPG
2. 在圖片編輯工具中打開
3. 調整為合適的打印尺寸
4. 打印到卡片紙上

或使用 HTML 版本直接打印。

## 技術問題

### Q29: 數據儲存在哪裡？
**A:** 所有數據：
- 儲存在您的瀏覽器本地儲存 (localStorage)
- 不上傳到任何服務器
- 刷新頁面後數據仍保留
- 清除瀏覽器數據時會被刪除

### Q30: 我的數據安全嗎？
**A:** 是的，非常安全：
- 所有處理在本地進行
- 無需傳送到服務器
- 您完全控制您的數據
- 無第三方訪問

### Q31: 瀏覽器需要什麼 JavaScript 支援？
**A:** 需要支援：
- ES6 JavaScript
- React 18
- CSS3 (Flexbox, Grid, Animations)
- HTML5 Canvas

所有現代瀏覽器都支援。

### Q32: 應用程式消耗多少網絡流量？
**A:** 
- 首次加載：~3-5MB（下載庫和資源）
- 日常使用：< 1MB（主要是圖片上傳）
- 後續訪問：最小（使用瀏覽器緩存）

### Q33: 如何優化性能？
**A:** 建議：
- 使用最新的瀏覽器版本
- 清除瀏覽器緩存
- 禁用不必要的擴展
- 使用快速的網絡連接
- 選擇簡潔的設計避免過度複雜

### Q34: 支援哪個 React 版本？
**A:** React 18.x。應用程式使用最新的 React Hooks API。

## 隱私和安全

### Q35: 應用程式收集哪些數據？
**A:** 不收集任何數據。
- 無追蹤代碼
- 無分析
- 無廣告
- 無登入系統

### Q36: 我的個人信息會被保存嗎？
**A:** 
- 信息只保存在您的瀏覽器
- 不傳送到任何服務器
- 刷新頁面後仍保留
- 清除瀏覽器數據時刪除

### Q37: 應用程式使用 Cookies 嗎？
**A:** 只使用本地 Cookies 保存設置（如主題偏好）。不用於追蹤。

### Q38: 導出的文件包含我的隱私數據嗎？
**A:** 是的。導出文件包含您輸入的所有信息。請僅與信任的人分享。

### Q39: 應用程式是否進行端到端加密？
**A:** 不需要，因為所有處理都在您的設備本地進行，數據不傳輸。

### Q40: 如何刪除我的所有數據？
**A:** 
1. 在瀏覽器中打開應用程式
2. 打開開發者工具 (F12)
3. 進入應用程式選項卡
4. 清除 localStorage
5. 或簡單地清除瀏覽器歷史和站點數據

## 疑難排除

### Q41: 為什麼頁面加載很慢？
**A:** 可能原因和解決方案：
- **網絡慢**：檢查連接速度
- **瀏覽器擴展**：禁用擴展
- **硬件不足**：關閉其他應用程式
- **緩存滿**：清除瀏覽器緩存

### Q42: 為什麼無法上傳圖片？
**A:** 檢查以下項目：
- 文件格式（JPG、PNG、WebP、GIF）
- 文件大小（< 5MB）
- 瀏覽器權限（允許文件訪問）
- 可用磁碟空間

### Q43: 為什麼導出失敗？
**A:** 常見原因：
- 瀏覽器不支援該格式
- 文件太大
- JavaScript 被禁用
- 彈出窗口被阻止

### Q44: 為什麼動畫不播放？
**A:** 檢查：
- 瀏覽器是否為最新版本
- CSS 動畫是否啟用
- 硬件加速是否啟用
- 選擇的動畫效果是否為「無」

### Q45: 為什麼某些功能不工作？
**A:** 嘗試：
1. 刷新頁面（Ctrl+Shift+R）
2. 清除瀏覽器緩存
3. 禁用擴展
4. 使用無痕窗口
5. 嘗試另一個瀏覽器
