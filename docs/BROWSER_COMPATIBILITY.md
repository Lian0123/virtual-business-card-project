## ⚠️ 已知問題和解決方案

### Safari 特定問題

#### 1. WebP 格式導出不支援
**症狀：** WebP 導出選項不可用或導出失敗
**原因：** Safari 對 WebP 的支援有限
**解決方案：** 使用 PNG 或 JPG 格式替代

#### 2. 某些濾鏡效果不工作
**症狀：** 濾鏡效果在 Safari 中顯示不完整
**原因：** CSS 濾鏡兼容性差異
**解決方案：** 使用基本的亮度和對比度設定

#### 3. 字體渲染差異
**症狀：** 文字在 Safari 中看起來不同
**原因：** 系統字體差異
**解決方案：** 使用系統安全字體或 Web 字體

### iOS 特定問題

#### 1. 文件下載限制
**症狀：** 無法直接下載文件到設備
**原因：** iOS 安全限制
**解決方案：** 使用共享菜單或 iCloud Drive

#### 2. 複製粘貼圖片
**症狀：** 無法直接複製卡片圖片
**原因：** iOS 限制
**解決方案：** 先導出為圖片格式，然後在相冊中操作

### Android 特定問題

#### 1. 大檔案導出
**症狀：** 導出大型高分辨率圖片時崩潰
**原因：** 內存限制
**解決方案：** 使用 WebP 格式或減小圖片尺寸

#### 2. 字體支援不完整
**症狀：** 某些字體顯示為方框
**原因：** 系統字體不完整
**解決方案：** 選擇基本字體或安裝字體包

## 🛠️ 故障排除指南

### 常見問題

#### 問題 1：頁面加載緩慢
**可能原因：**
- 網絡連接不穩定
- 瀏覽器擴展影響
- 系統資源不足

**解決步驟：**
1. 重新加載頁面（F5 或 Ctrl+Shift+R）
2. 禁用瀏覽器擴展
3. 清除瀏覽器緩存
4. 在無痕窗口中嘗試

#### 問題 2：圖片上傳失敗
**可能原因：**
- 圖片格式不支援
- 文件大小過大
- 瀏覽器權限問題

**解決步驟：**
1. 確認圖片格式（JPG、PNG、WebP）
2. 調整圖片大小（< 5MB）
3. 檢查瀏覽器權限
4. 在另一個瀏覽器中嘗試

#### 問題 3：導出功能不工作
**可能原因：**
- 瀏覽器不支援
- JavaScript 被禁用
- 彈出窗口被阻止

**解決步驟：**
1. 檢查瀏覽器版本
2. 在設置中啟用 JavaScript
3. 允許彈出窗口
4. 清除瀏覽器緩存

#### 問題 4：動畫不播放
**可能原因：**
- CSS 動畫不支援
- 瀏覽器禁用了動畫
- 硬件加速問題

**解決步驟：**
1. 更新瀏覽器到最新版本
2. 檢查瀏覽器動畫設置
3. 禁用瀏覽器擴展
4. 嘗試另一個瀏覽器

## 📋 兼容性檢查清單

在使用應用程式之前，確認您已滿足以下條件：

- [ ] 瀏覽器已更新到最新版本
- [ ] JavaScript 已啟用
- [ ] Cookie 已啟用（用於保存設置）
- [ ] localStorage 已啟用
- [ ] 有足夠的磁碟空間用於下載
- [ ] 網絡連接穩定
- [ ] 無阻止性的瀏覽器擴展

   - 操作系統
