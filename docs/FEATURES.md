## 基本資訊

### 個人信息輸入

所有基本信息都在「基本資訊」選項卡中：

| 欄位 | 說明 | 必填 |
|------|------|------|
| 名字 | 您的全名 | ✅ |
| 職位 | 您的職務 | ✅ |
| 公司 | 所在公司名稱 | ✅ |
| 郵箱 | 電子郵件地址 | ✅ |
| 電話 | 聯繫電話號碼 | ✅ |
| 國家 | 所在國家 | ❌ |
| 地址 | 完整地址 | ❌ |
| 網站 | 個人或公司網站 | ❌ |
| 簡介 | 自我介紹 | ❌ |

### 頭像管理

- **上傳頭像**：點擊頭像框選擇本地圖片
- **支援格式**：JPG、PNG、GIF、WebP
- **推薦大小**：300x300px 到 600x600px
- **最大大小**：5MB

### 頭像形狀

選擇適合您的頭像形狀：
- 🔵 **圓形** - 現代、友善的外觀
- ⬜ **方形** - 專業、正式的外觀  
- ⬛ **圓角方形** - 平衡、溫和的外觀

## 卡片類型

### 1. 標準卡片（Main Card）

最常用的卡片類型，展示您的基本信息。

**佈局：**
```
┌─────────────────────────────────┐
│ 頭像 │  名字              │
│     │  職位 @ 公司       │
│     │  ────────────────  │
│     │  國家、地址        │
│     │  ────────────────  │
│     │  📧 郵箱           │
│     │  📱 電話           │
│     │  🌐 網站           │
└─────────────────────────────────┘
```

**最佳用途：**
- 名片交換
- 電子郵件簽名
- 社交媒體

### 2. 二維碼卡片（QR Card）

包含自動生成的二維碼，方便他人快速訪問您的信息。

**佈局：**
```
┌──────────────────────────────────┐
│ 名字                │ ┌─────────┐ │
│ 職位                │ │ QR Code │ │
│ 簡介...             │ │         │ │
│ 郵箱、電話          │ └─────────┘ │
└──────────────────────────────────┘
```

**二維碼包含：**
- 您的名字
- 郵箱地址
- 電話號碼
- 或自訂內容

**最佳用途：**
- 名片底部
- 社交媒體個人資料
- 事件簽到

### 3. 自訂卡片（Custom Card）

完全自由設計，支援自訂背景和貼圖。

**特性：**
- 自訂背景色或背景圖片
- 自由添加文字和圖片貼圖
- 完全控制佈局和位置
- 無限創意可能

**最佳用途：**
- 創意設計
- 品牌特色卡片
- 特殊用途設計

## 視覺自訂

### 背景設定

#### 背景顏色
- 選擇單一顏色作為背景
- 支援所有 RGB 顏色（16 百萬種）

#### 背景漸層
1. 選擇「漸層色彩 1」（起始顏色）
2. 選擇「漸層色彩 2」（終止顏色）
3. 漸層方向：135 度角（左上到右下）
4. 實時預覽效果

#### 背景圖片
- 上傳自訂背景圖片
- 支援 JPG、PNG、WebP
- 自動覆蓋背景顏色

**建議：**
- 使用淺色背景確保文字可讀
- 高分辨率圖片以獲得最佳效果
- 考慮在深色圖片上使用淺色文字

### 邊框設定

#### 邊框寬度
- 範圍：0-20px
- 0px = 無邊框
- 推薦：2-4px

#### 邊框模式

**純色模式：**
- 選擇單一顏色
- 簡潔、經典的外觀
- 點擊色彩選擇器即可選色

**漸層模式：**
- 選擇兩種顏色進行漸層
- 漸層色彩 1：起始顏色
- 漸層色彩 2：終止顏色
- 135 度角漸層效果

### 陰影效果

調整卡片的陰影參數：

| 參數 | 範圍 | 說明 |
|------|------|------|
| 陰影 X | -20 to 20 | 水平偏移 |
| 陰影 Y | -20 to 20 | 垂直偏移 |
| 陰影模糊 | 0-50 | 陰影柔和度 |
| 陰影顏色 | 所有顏色 | 陰影顏色 |

**預設值：**
- X = 0
- Y = 8px
- 模糊 = 24px
- 顏色 = 黑色 (15% 透明)

### 濾鏡效果

應用各種視覺效果：

| 濾鏡 | 範圍 | 預設值 | 說明 |
|------|------|--------|------|
| 亮度 | 0-200% | 100% | 增加或減少亮度 |
| 對比度 | 0-200% | 100% | 增加或減少對比度 |
| 飽和度 | 0-200% | 100% | 增加或減少色彩飽和度 |
| 色調 | 0-360° | 0° | 旋轉色輪 |
| 銳度 | 0-100% | 0% | 增加邊界銳度 |
| 模糊 | 0-50px | 0px | 高斯模糊效果 |

## 進階功能

### 動畫效果

支援 20+ 種 Animate.css 動畫：

#### 進入動畫
- `fadeIn` - 淡入
- `slideInLeft` - 從左滑入
- `slideInRight` - 從右滑入
- `slideInUp` - 從下滑入
- `slideInDown` - 從上滑入

#### 關注動畫
- `bounce` - 彈跳
- `pulse` - 脈衝
- `shake` - 搖晃
- `swing` - 搖擺
- `tada` - 驚喜

#### 旋轉動畫
- `rotateIn` - 旋轉進入
- `rotateOut` - 旋轉退出
- `flip` - 翻轉

#### 其他動畫
- `zoomIn` - 縮放進入
- `zoomOut` - 縮放退出
- `lightSpeedIn` - 光速進入

### 動畫控制

#### 播放控制按鈕（左上角）
- **▶ 播放** - 開始或恢復動畫
- **⏸ 暫停** - 暫停當前動畫
- **🔄 重播** - 重新開始動畫

#### 動畫設定
- 動畫效果：選擇您喜歡的動畫
- 動畫時長：1 秒（固定）
- 自動循環：持續重複播放

### 貼圖功能

#### 文字貼圖

**添加步驟：**
1. 在「貼圖編輯」選項卡中
2. 點擊「添加文字貼圖」
3. 輸入您的文字內容

**文字樣式：**
- 🅰️ **純文字** - 標準文字
- 💫 **陰影** - 添加投影效果
- 🔤 **描邊** - 添加文字輪廓
- **粗體** - 加粗文字
- *斜體* - 傾斜文字

**文字控制：**
- 位置：X% 和 Y%（左上角為 0,0）
- 大小：1-100px
- 顏色：選擇任何顏色
- 透明度：0-100%
- 旋轉：0-360°

#### 圖片貼圖

**添加步驟：**
1. 點擊「上傳貼圖圖片」
2. 選擇您的圖片文件
3. 調整大小和位置

**圖片控制：**
- 位置：X% 和 Y%
- 寬度和高度：自訂尺寸
- 透明度：0-100%
- 旋轉：0-360°

**支援格式：**
- JPG、PNG、GIF、WebP
- 最大大小：10MB

## 導出選項

### 圖片格式

#### PNG 格式 🎨
```
優點：無損壓縮、高質量、透明背景支援
用途：網頁、社交媒體、打印
檔案大小：中等（通常 200-500KB）
```

#### JPG 格式 📸
```
優點：廣泛支援、檔案較小
用途：郵件、網站、打印
檔案大小：小（通常 50-200KB）
缺點：無透明度支援
```

#### WebP 格式 ⚡
```
優點：最小檔案大小、高質量
用途：現代網頁、移動設備
檔案大小：最小（通常 30-100KB）
支援：現代瀏覽器（IE 不支援）
```

### HTML 格式 🌐

**完整的網頁版本：**
- 獨立的 HTML 檔案
- 包含所有樣式和動畫
- 可在任何瀏覽器中打開
- 支援 QR 碼自動生成
- 點擊卡片可暫停/播放動畫

**用途：**
- 嵌入個人網站
- 郵件中嵌入
- 在線投資組合
- 動畫展示

### VCF 格式 📇

**標準聯繫卡片：**
- 通用聯繫人格式
- 可導入手機通訊錄
- 包含所有基本信息

**用途：**
- 分享聯繫方式
- 手機聯繫人導入
- 郵件簽名

## 性能和優化

### 檔案大小參考

| 格式 | 典型大小 | 最佳用途 |
|------|--------|--------|
| PNG | 200-500KB | 高質量、無損 |
| JPG | 50-200KB | 一般網絡 |
| WebP | 30-100KB | 移動設備 |
| HTML | 100-300KB | 動畫展示 |
| VCF | < 10KB | 聯繫人 |

### 優化建議

- 使用 WebP 以獲得最小檔案大小
- 選擇簡潔的動畫避免過度複雜
- 優化圖片大小在上傳前
- 使用對比色提升視覺效果

## 提示和技巧

### 設計技巧
✅ 使用互補色搭配
✅ 保持設計簡潔
✅ 確保文字清晰易讀
✅ 測試在不同設備上的效果

- 在導出前進行充分測試
- 在多個瀏覽器中檢查
- 考慮訪問者的用戶體驗
