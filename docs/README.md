## 🚀 快速開始

### 最簡單的方式
1. 在瀏覽器中打開 `index.html`
2. 填寫您的個人信息
3. 選擇喜歡的設計範本
4. 點擊「匯出」選擇格式

### 所有版本支援的導出格式
- **PNG** - 高質量圖片格式
- **JPG** - 標準圖片格式
- **WebP** - 現代圖片格式
- **HTML** - 可互動的網頁版本
- **VCF** - 聯繫人卡片格式

## 💡 使用技巧

### 優化性能
1. 使用 WebP 格式以獲得更小的檔案大小
2. 為網絡使用優化圖像大小
3. 選擇合適的動畫效果避免過度複雜

### 最佳實踐
1. 始終備份您的設計
2. 在匯出前預覽結果
3. 使用高質量的頭像圖片
4. 測試不同瀏覽器的相容性

### 故障排除
- 若圖片未加載：檢查檔案格式和大小
- 若動畫不播放：更新您的瀏覽器
- 若無法匯出：清除瀏覽器快取
