# 快速開始指南

## 👋 歡迎！

本指南將幫助您在 5 分鐘內創建第一張電子名片。

---

## 第一步：準備就緒

### 您需要的信息：
- 📝 您的名字
- 💼 職位和公司名稱
- 📧 郵箱地址
- 📱 電話號碼
- 🌐 網站（可選）
- 📍 國家和地址（可選）
- 🖼️ 頭像照片（推薦 300x300px）

---

## 第二步：打開應用程式

1. 在您的網絡瀏覽器中打開 `index.html`
2. 頁面加載完成後，您將看到左側的控制面板和右側的預覽

```
┌─────────────────────────────────────────┐
│   控制面板              │    預覽區域   │
│  (基本信息、設置)      │   (實時預覽)  │
└─────────────────────────────────────────┘
```

---

## 第三步：填寫基本信息

### 在左側面板中：
1. **基本資訊** 選項卡
   - 輸入您的名字
   - 輸入您的職位
   - 輸入您的公司名稱
   - 輸入您的郵箱
   - 輸入您的電話

2. **上傳頭像**
   - 點擊頭像框上傳您的照片
   - 選擇頭像形狀（圓形/方形/圓角）

3. **卡片類型**
   - 選擇 "標準卡片" 開始使用

✅ **結果：** 您應該在右側看到卡片預覽已更新

---

## 第四步：美化設計

### 在「進階設定」選項卡中：

1. **選擇設計範本**
   - 點擊任意範本立即應用
   - 可用範本：專業、現代、極簡、活力、日落、海洋、森林

2. **自訂背景**
   - 背景顏色：選擇單一顏色
   - 背景漸層色彩 1 和 2：創建漸層效果

3. **設定邊框**
   - 邊框寬度：0-20px
   - 邊框模式：純色或漸層
   - 邊框顏色：選擇適合的顏色

4. **添加效果**
   - 動畫效果：選擇喜歡的動畫
   - 濾鏡效果：調整亮度、對比度等

✅ **技巧：** 實時預覽會立即顯示所有更改

---

## 第五步：動畫控制

### 左上角的控制按鈕：
- **▶ 播放** - 開始播放動畫
- **⏸ 暫停** - 暫停當前動畫
- **🔄 重播** - 重新播放動畫

---

## 第六步：添加貼圖（可選）

### 添加文字貼圖：
1. 進入「貼圖編輯」選項卡
2. 點擊「添加文字貼圖」
3. 輸入您的文字
4. 調整樣式（粗體、斜體、陰影等）
5. 設定位置、大小、顏色

### 添加圖片貼圖：
1. 點擊「上傳貼圖圖片」
2. 選擇您的圖片
3. 調整大小和位置

---

## 第七步：匯出名片

### 選擇匯出格式：

#### 📷 圖片格式（PNG、JPG、WebP）
```
優點：易於分享、適合打印、支援所有設備
使用場景：社交媒體、郵件簽名、打印
```

#### 🌐 HTML 格式
```
優點：互動性強、可嵌入網頁、支援動畫
使用場景：個人網站、在線投資組合
```

#### 📇 VCF 格式
```
優點：標準聯繫卡片格式
使用場景：導入手機聯繫人、發送給他人
```

### 匯出步驟：
1. 點擊右側預覽區下方的「匯出」按鈕
2. 選擇您喜歡的格式
3. 檔案將自動下載

---

## 實用技巧

### 💡 設計技巧
- 使用互補色提升視覺效果
- 避免過多的動畫效果
- 確保文字清晰易讀
- 測試不同設備上的效果

### ⚙️ 性能技巧
- 使用 WebP 格式以獲得更小檔案
- 優化圖片大小（推薦 < 2MB）
- 選擇簡潔的動畫效果

### 🔄 反覆調整
- 所有更改都是實時的
- 無需保存，直接查看效果
- 可以隨時改變想法

## 常見問題快速解答

**Q: 如何更改語言？**
A: 點擊右上角的語言選擇器

**Q: 頭像無法上傳？**
A: 檢查圖片格式（建議 JPG/PNG）和大小（< 5MB）

**Q: 如何預覽 HTML 匯出的效果？**
A: 下載後在任何瀏覽器中打開 HTML 檔案

**Q: 可以編輯已匯出的檔案嗎？**
A: HTML 檔案可以編輯，重新刷新應用程式重新設計
