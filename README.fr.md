# Virtual Business Card Project

Créez, éditez, prévisualisez et exportez des cartes de visite numériques professionnelles directement dans le navigateur. Prise en charge de PWA, du contrôle distant WebMCP (JSON-RPC) et des formats PNG/JPG/WebP/HTML/VCF.

## Fonctionnalités clés
- Édition de carte: nom, poste, entreprise, contacts, adresse, site, présentation
- Design visuel: thème, modèle, fond, bordure, animation, stickers
- Export: PNG / JPG / WebP / HTML / VCF
- Intégration IA: WebMCP (`tools/list`, `tools/call`, `card.*`)
