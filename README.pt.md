# Virtual Business Card Project

Crie, edite, visualize e exporte cartões de visita digitais profissionais diretamente no navegador. Suporta PWA, controlo remoto via WebMCP (JSON-RPC) e exportação em PNG/JPG/WebP/HTML/VCF.

## Funcionalidades principais
- Edição do cartão: nome, cargo, empresa, contactos, morada, website, apresentação
- Design visual: tema, modelo, fundo, borda, animação, stickers
- Exportação: PNG / JPG / WebP / HTML / VCF
- Integração com IA: WebMCP (`tools/list`, `tools/call`, `card.*`)
