# Virtual Business Card Project

Twórz, edytuj, podglądaj i eksportuj profesjonalne cyfrowe wizytówki bezpośrednio w przeglądarce. Obsługa PWA, zdalnego sterowania przez WebMCP (JSON-RPC) oraz formatów PNG/JPG/WebP/HTML/VCF.

## Najważniejsze funkcje
- Edycja karty: imię, stanowisko, firma, kontakt, adres, strona, opis
- Projekt wizualny: motyw, szablon, tło, obramowanie, animacje, naklejki
- Eksport: PNG / JPG / WebP / HTML / VCF
- Integracja AI: WebMCP (`tools/list`, `tools/call`, `card.*`)
