# Virtual Business Card Project

Buat, edit, pratinjau, dan ekspor kartu nama digital profesional langsung dari browser. Mendukung PWA, kontrol jarak jauh WebMCP (JSON-RPC), dan format ekspor PNG/JPG/WebP/HTML/VCF.

## Fitur Utama
- Edit kartu: nama, jabatan, perusahaan, kontak, alamat, situs, intro
- Desain visual: tema, template, latar belakang, border, animasi, stiker
- Ekspor: PNG / JPG / WebP / HTML / VCF
- Integrasi AI: WebMCP dengan `tools/list`, `tools/call`, dan tool `card.*`
