# Virtual Business Card Project

Crea, edita, previsualiza y exporta tarjetas de presentación digitales profesionales directamente en el navegador. Compatible con PWA, control remoto WebMCP (JSON-RPC) y exportación PNG/JPG/WebP/HTML/VCF.

## Funciones principales
- Edición de tarjeta: nombre, cargo, empresa, contacto, dirección, web, presentación
- Diseño visual: tema, plantilla, fondo, borde, animación, stickers
- Exportación: PNG / JPG / WebP / HTML / VCF
- Integración con IA: WebMCP (`tools/list`, `tools/call`, `card.*`)
